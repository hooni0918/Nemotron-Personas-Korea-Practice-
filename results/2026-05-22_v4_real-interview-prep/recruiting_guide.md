# 실사용자 인터뷰 모집 가이드

## 먼저 — 이 문서는 뭔가요

장소 스크랩 서비스 인터뷰에 참여할 사람을 5~8명 모집하는 가이드입니다. 시뮬레이션 결과(v3)와 비교하기 위해, 시뮬과 같은 페르소나 분포로 맞춰서 뽑는 게 중요합니다.

---

## 누구를 몇 명 모집하나

| 라벨 | 누구인가 | 시뮬 비율 | 실제 인터뷰 권장 |
|---|---|---|---|
| **A_marketer** | 마케팅·콘텐츠·디자인 직군 | 33% (40/120명) | **2~3명** — 베타 타겟 1순위, 그룹 기능에 가장 강한 반응 |
| **B_engineer** | IT·개발 직군 | 25% (30/120명) | **1~2명** — 자동 저장에 강한 반응 (효율 중시) |
| **C_general** | 일반 직군 (사무직·서비스직 등) | 42% (50/120명) | **2~3명** — 일반 사용자 베이스라인 |
| **합계** |  |  | **5~8명** (5명이 최소, 8명이면 충분) |

A_marketer를 가장 많이 모집하는 이유는, 시뮬에서 이 그룹이 가장 강한 시그널을 보였기 때문입니다. **"시뮬에서 강한 패턴이 실제로도 강한가"** 를 확인하는 게 가장 가치 있는 검증이에요.

---

## 모집 조건

### 공통 조건 (모든 응답자)

- [ ] 만 20~39세
- [ ] 수도권·광역시 거주 (서울·경기·인천·부산·대구·대전·광주·울산·세종)
- [ ] **인스타그램을 일주일에 3번 이상 사용**
- [ ] **최근 한 달 안에 인스타에서 가고싶은 장소를 발견한 경험이 1번 이상**

### 라벨별 추가 조건

**A_marketer 모집 시**
- [ ] 마케팅·광고·홍보·브랜드·디자인·콘텐츠·미디어·기획 관련 직군
- [ ] 직장인 또는 프리랜서 (학생 제외)

**B_engineer 모집 시**
- [ ] IT·개발·데이터·엔지니어링 관련 직군
- [ ] 직장인 또는 프리랜서

**C_general 모집 시**
- [ ] 위 두 그룹에 해당하지 않는 직군 (사무직·서비스직·자영업 등)
- [ ] 인스타 활발 사용은 동일

---

## 스크리닝 질문 (모집 폼이나 사전 통화에서)

먼저 다음 5문항을 빠르게 묻고, 통과한 분만 본 인터뷰 일정을 잡으세요.

```
1. 나이가 어떻게 되시나요?
   → 20~39세만 통과

2. 어디 사세요?
   → 수도권·광역시만 통과

3. 무슨 일 하세요?
   → 직업 듣고 A / B / C 라벨로 분류

4. 인스타그램은 일주일에 몇 번 정도 쓰세요?
   → 3번 이상만 통과

5. 최근 한 달 안에 인스타에서 "여기 가보고 싶다" 싶은 장소(맛집·카페·팝업 등)를
   발견한 적이 있나요? 있다면 그때 어떻게 하셨는지 간단히 말씀해 주세요.
   → "있다 + 어떤 행동(저장·캡처·DM 등)"이라고 답해야 통과
   → "없다" 또는 "그냥 넘긴다"고 답하면 비대상
```

5번 질문이 가장 중요합니다. 시뮬 페르소나는 모두 "저장 경험 있음"이 전제인데, 실제 사용자가 저장 경험 자체가 없으면 비교가 의미 없어요.

## 모집 채널 후보

| 채널 | 어느 라벨 모집에 좋은가 | 비고 |
|---|---|---|
| 마케팅 직군 디스코드·뉴스레터 | A | "그로스 마케터 모임", "퍼블리" 같은 곳 |
| 디자이너 슬랙·커뮤니티 | A | "디자이너 코리아" 등 |
| 개발자 커뮤니티 | B | OKKY, 디스코드 — 생각보다 인스타 활발한 분 많아요 |
| 본인 지인 1차 추천 | A·B·C 모두 | 가장 빠릅니다. 단 인스타 활발 여부는 꼭 사전 확인 |
| 리워드 모집 (커피쿠폰 1만원) | C | 일반 사용자 모집할 때 |
| 모집 서비스 (테스트밋·유저블라인드) | 전체 | 비용 드는 대신 시간 절약 |

---

## 사례비 권장

- **커피쿠폰 1만원** — 가벼운 부담, 지인 모집 시 충분
- **현금/기프티콘 3만원** — 모집 서비스 통해 일반 모집 시
- **5만원 이상** — 모집 어려운 직군이거나 1시간 풀 인터뷰일 때

---

## 인터뷰 형식

- **소요 시간**: 60~75분
- **형식**: 대면 또는 화상 (Zoom·구글미트)
- **녹음**: 시작 전 동의 받기 (분석할 때 필수)
- **준비물**: 인터뷰 가이드([`interview_guide_v1.md`](./interview_guide_v1.md)), 와이어프레임 (있다면)

---

## 모집 메시지 템플릿

인스타·디스코드·커뮤니티에 그대로 올려도 됩니다.

```
[페이드 인터뷰 모집] 인스타 핫플 저장 습관에 대해 이야기 나눠요

새로운 장소 저장 앱을 만들고 있는데,
인스타에서 가고싶은 곳을 자주 발견하시는 분들의 솔직한 의견을 듣고 싶어요.

✅ 대상: 20~39세, 인스타 일주일 3번 이상 사용, 최근 한 달 안에 저장·공유 경험 있는 분
✅ 소요: 1시간 화상 인터뷰
✅ 사례: 3만원 기프티콘

마케팅·콘텐츠·디자인 직군 분 우대.
관심 있으신 분은 DM 또는 폼 [링크] 작성 부탁드립니다 🙏
```
