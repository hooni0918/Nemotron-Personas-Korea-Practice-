# 시뮬 vs 실제 인터뷰 비교 분석 템플릿

## 먼저 — 이 문서는 어떻게 쓰나요

인터뷰 5~8명 끝난 뒤에 이 문서를 열고 빈칸을 채우면 됩니다. 각 질문마다 칸이 4개 있어요.

1. **시뮬 결과** — 미리 채워뒀습니다 (v3 120명 결과)
2. **실제 응답** — 본인이 채울 빈칸 (P1, P2, P3 형식)
3. **🟢🟡🔴 라벨** — 시뮬과 실제가 얼마나 일치하는지 한 글자로
4. **디자이너·기획자 take-away** — 한 줄로 "그래서 어떻게 하면 좋을까"

다 채우면 그대로 v5 보고서가 됩니다.

---

## 가장 빠른 길로 채우는 5단계

1. 인터뷰 노트나 transcript 펴두세요.
2. 응답자별로 P1, P2... 번호를 매기세요 (아래 명단 표에).
3. 위에서 아래로 질문 순서대로 한 칸씩 채우세요. 응답자별로 채우려고 하지 마세요 (질문별로 가는 게 빠릅니다).
4. 칸을 채우면서 직감적으로 🟢🟡🔴 중 하나 고르세요. 기준은 아래 라벨링 가이드 참고.
5. 마지막에 맨 아래 **종합 의사결정 매트릭스**까지 채우면 끝입니다.

대략 1시간이면 끝납니다.

## 라벨링 가이드 — 🟢🟡🔴

직감으로 빨리 골라도 됩니다. 정확한 기준:

| 라벨 | 어떤 경우인가 | 디자이너·기획자에게 어떤 의미 |
|---|---|---|
| 🟢 일치 | 시뮬 분포랑 실제 응답이 거의 같은 방향 | 시뮬 결과를 정량 근거로 자신 있게 인용해도 됩니다 |
| 🟡 부분 일치 | 큰 방향은 같은데 비율이나 강도가 다름 | 시뮬은 참고만 하고, 결정은 실제 응답으로 |
| 🔴 다름 | 시뮬과 실제가 반대 방향이거나 완전히 다름 | 시뮬 결과 폐기하고 실제 인터뷰만 보세요 |

빠른 기준:
- 시뮬 87% + 실제 5명 모두 같은 방향 → 🟢
- 시뮬 87% + 실제 3/5명 → 🟡
- 시뮬 87% + 실제 1/5명 → 🔴
