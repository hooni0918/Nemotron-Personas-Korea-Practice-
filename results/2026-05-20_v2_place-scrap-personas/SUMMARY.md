## 🎯 핵심 검증 결과: 페르소나 분화가 매우 명확하게 작동

PDF에서 의도한 **3 타입의 행동·페인 차이가 그대로 분화됨**. 척도 질문 응답을 키워드 카운트한 결과:

### Q3 — 저장한 장소 방문률 (5명/타입)

| 페르소나 | 대부분 방문 | 가끔 방문 | 거의/전혀 안 함 |
|---|---|---|---|
| **실행형** | **5** | 0 | 0 |
| **수집형** | 0 | 0 | **5** |
| **즉흥형** | 0 | **5** | 0 |

→ PDF 설계대로 완벽히 분리. **"저장 잘 함 → 방문 잘 감 / 안 감 / 안 함"** 분류가 LLM에서 일관되게 재현됨.

### Q15 — "왜 저장했는지" 헷갈린 경험

| 페르소나 | 자주 있다 | 가끔 있다 | 거의/없 |
|---|---|---|---|
| 실행형 | 3 | 2 | 0 |
| **수집형** | **5** | 0 | 0 |
| 즉흥형 | 0 | 3 | 3 |

→ **수집형 5/5 "자주" — 핵심 페인의 정수**. 이게 곧 1순위 타겟인 이유.

### Q13 — MVP1 "링크 자동 저장" 편의도

| 페르소나 | 매우 편할 | 조금 더 편할 | 비슷/불편 |
|---|---|---|---|
| **실행형** | **4** | 0 | 0 |
| **수집형** | **4** | 0 | 0 |
| 즉흥형 | 0 | **4** | 0 |

→ 저장형 둘 다 압도적 긍정. 즉흥형은 "나쁘지 않다" 톤. **MVP1 가치 명제 검증**.

### Q5 — 저장한 장소를 다시 찾을 때 불편

| 페르소나 | 매우 자주 | 자주 | 가끔 이하 |
|---|---|---|---|
| 실행형 | 5 | 5 | 0 |
| 수집형 | 5 | 5 | 0 |
| 즉흥형 | 3 | 3 | 5 |

→ 저장형은 강한 페인, 즉흥형은 절반 정도. **재탐색 페인이 저장 습관 강도와 직접 연동됨**.

### Q18 — 친구/그룹 공동 저장 기능 필요성

| 페르소나 | 매우 그렇다 | 그렇다 | 보통/아니다 |
|---|---|---|---|
| **실행형** | **5** | (포함) | 0 |
| 수집형 | 1 | 4 | 1 |
| 즉흥형 | 2 | 2 | 4 |

→ **실행형이 그룹 기능을 가장 강하게 원함** (이미 친구와 약속을 잘 잡는 행동 패턴 때문). 의외로 수집형보다 강함.

## ⚠️ 진단된 질문 (자동 sentiment 분석 한계 표시)

분석기가 6개 질문(Q3·Q5·Q10·Q12·Q13·Q14)을 "중립 70%↑ → 모호" 진단함. 그러나 **실제 답변은 명확한 척도 응답**이고, sentiment 패턴 매칭이 5점 척도 응답을 "중립"으로 분류하는 한계 때문임. **진단기 자체의 약점**으로 기록 — v3에서 척도 응답 전용 분류기 추가 검토.
