## 2. 무엇을 물었고 어떻게 답했나

각 질문 + 응답 분포 (모두 120명 응답).

### 섹션 1. SNS 저장 행동

#### Q1. SNS에서 장소 콘텐츠를 저장/공유한 경험이 얼마나 있나요?

| 보기 | 응답 |
|---|---|
| 자주 한다 | 45 (37%) |
| 매우 자주 한다 | 14 (11%) |
| 가끔 한다 | 12 (10%) |

→ **저장 행동은 보편적** (자주+매우자주 49%).

#### Q2. 주로 어떤 방식으로 저장/공유하나요? (복수)

| 보기 | 응답 |
|---|---|
| 인스타 저장 | 73 (60%) |
| 캡처 | 17 (14%) |
| 메모 | 6 (5%) |
| DM 공유 | 3 (2%) |
| 지도 앱 저장 | 1 (0%) |

→ **인스타 저장 버튼 압도적 1위**.

#### Q3. 저장한 장소를 실제로 방문한 경험이 얼마나?

| 보기 | 응답 |
|---|---|
| 가끔 방문한다 | 100 (83%) |
| 거의 방문하지 않는다 | 7 (5%) |
| 대부분 방문한다 | **0** |

→ **저장 ≠ 방문** 격차 명확. "대부분 방문" 응답 0건.

#### Q4. 방문하지 않는 가장 큰 이유? (복수)

| 보기 | 응답 |
|---|---|
| 잊어버려서 | 86 (71%) |
| 다시 찾기 귀찮아서 | 56 (46%) |
| 저장만 하고 행동하지 않아서 | 54 (45%) |
| 누구와 가려고 했는지 잊어서 | 4 (3%) |
| 다른 장소를 가게 돼서 | 4 (3%) |
| 위치/동선이 애매 | 3 (2%) |

→ **"잊어버린다"** + **"다시 찾기 귀찮다"** = 핵심 페인.

### 섹션 2. 저장 이후 휘발/재탐색 문제

#### Q5. 저장한 장소를 다시 찾을 때 불편함?

| 보기 | 응답 |
|---|---|
| 자주 있다 | 73 (60%) |
| 가끔 있다 | 19 (15%) |
| 매우 자주 있다 | 17 (14%) |

→ **재탐색 불편 = 보편 페인** (매우 자주+자주 = 75%).

#### Q6. 다시 못 찾아서 포기한 경험?

| 보기 | 응답 |
|---|---|
| 자주 있다 | 71 (59%) |
| 가끔 있다 | 21 (17%) |
| 거의 없다 | 4 (3%) |
| 없다 | 2 (1%) |

→ **포기 경험 자주 있다 59%** — 실제 비즈니스 손실로 이어지는 행동.

#### Q7. 다시 찾기 어려운 가장 큰 이유? (복수)

| 보기 | 응답 |
|---|---|
| 왜 저장했는지(맥락) 까먹는다 | 84 (70%) |
| 왜 저장했는지 까먹는다 (변형 표현) | 80 (66%) |
| 어디에 저장했는지 기억이 안 난다 | 44 (36%) |
| 대화방/플랫폼이 헷갈린다 | 16 (13%) |
| 링크가 삭제됐다 | 15 (12%) |
| 저장은 했지만 다시 보지 않는다 | 11 (9%) |
| 스크롤을 많이 해야 한다 | 6 (5%) |

→ **"맥락 망각"이 압도적 1위 (70%)**. 다시 찾기 어려운 이유는 위치/스크롤이 아니라 "왜 저장했는지 기억 안 남".

#### Q8. 이전에 봤던 장소를 어떤 기준으로 기억하나? (복수)

| 보기 | 응답 |
|---|---|
| 콘텐츠 분위기/사진 | 68+65 (54~56%) |
| 누구와 공유했는지 | 51 (42%) |
| 위치/지역 | 44+37 (30~36%) |
| 어느 플랫폼에서 봤는지 | 11 (9%) |
| 시간/시기 | 2 (1%) |

→ **시각 기억(분위기·사진) > 사람 기억 > 위치 기억 > 플랫폼 기억** 순.

### 섹션 3. 현재 지도 앱/저장 방식의 한계

#### Q9. 지도 앱에 저장할 때 가장 불편한 점? (복수)

| 보기 | 응답 |
|---|---|
| 장소가 많아지면 관리 어려움 | 43 (35%) |

| 인스타 감성/정보 부족 | 12+15 (10~12%) |

→ **관리 어려움 + 저장 번거로움**이 양대 페인.

### 섹션 4. 친구 협업

#### Q10. 친구와 장소 정할 때 가장 불편? (복수)

### 섹션 5. MVP 기능 검증

| 오히려 불편할 것 같다 | 0 |

→ **MVP1 핵심 가치 87% 만장일치 긍정**.

#### Q14. (비슷/불편이라 답했다면) 그 이유는?

| 링크 기반 저장 불편 | 3 (2%) |

→ "비슷/불편" 답이 거의 없었지만, 잠재 우려는 **앱 전환 마찰**과 **인스타 감성 손실**.
